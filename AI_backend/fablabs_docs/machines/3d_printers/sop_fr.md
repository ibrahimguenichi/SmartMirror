# SOP - Imprimantes 3D

## Préparation
- Profil slicer conforme (Cura/Prusa)
- Filament approuvé (PLA/ABS selon ventilation)

## Paramètres
- Hauteur couche: 0.2 mm (par défaut)
- Température buse: selon filament

## Post-traitement
- Retrait supports, nettoyage plateau
